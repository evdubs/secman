Create Assignment

Creates an asset assignment. An assignment is an allocation of an amount of the asset to an investor. It is possible to set the value 'ready_for_distribution' to true (false), so that the assignment will (not) be included in future distributions. It is possible to specify a 'vesting_timestamp', so that the assignment will be included only in distributions created after the corresponding date.
Vesting timestamp must be a future timestamp, may be null. Assignment will not be distributed before the corresponding date.
