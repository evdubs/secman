Asset Activities

Returned activities can be of type: issuance, reissuance, distribution, transactions. The start parameter is 1 (not zero) based to make paging easier for clients. A negative value can be provided to return from the end of the list. Height_start and height_stop will filter and return only activites confirmed between these block heights. If there are lost outputs, we will include them in this list.
